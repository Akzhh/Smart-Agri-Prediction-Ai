🌱 AgriPredict AI: System Documentation

AgriPredict AI is an Explainable Machine Learning (XML) application that advises agricultural producers, agronomists, and soil researchers on optimal crop selection utilizing real-time soil nutrient profiling and environmental criteria.

📐 Core Architecture & Components

  • Vite React Frontend: A modern, modular single-page interface leveraging dynamic data visualizations (Recharts) and motion pathways (Framer Motion).
  • Flask REST API: A high-throughput microservice interface exposing analytical machine learning model pipelines.
  • Random Forest Classifier: A serialized multi-class decision boundary configuration processing complex environment parameters.

🤖 Machine Learning Pipeline & Dataset

The system uses 7 key input features to determine crop viability:

Feature Name Variable Code Unit Relevance
Nitrogen (N) N mg/kg Responsible for leaf growth and plant health.
Phosphorus (P) P mg/kg Critical for root development and flowering.
Potassium (K) K mg/kg Improves overall crop quality and water regulation.
Temperature temperature °C Checks suitability to specific regional climates.
Humidity humidity % Assesses atmospheric moisture thresholds.
Soil pH ph pH units Dictates nutrient solubility and uptake rates.
Rainfall rainfall mm Hydrological profile of the cultivation field.

🔌 API Specification

POST /predict

Yields crop classification distributions and explanation parameters.

Request Schema:

{
  "N": 90,
  "P": 42,
  "K": 43,
  "temperature": 20.8,
  "humidity": 82,
  "ph": 6.5,
  "rainfall": 202.5
}

Response Schema:

{
  "recommendation": "Rice",
  "top_recommendations": [
    { "crop": "Rice", "probability": 0.88 },
    { "crop": "Jute", "probability": 0.08 }
  ],
  "feature_importance": {
    "N": 0.125,
    "P": 0.085,
    "K": 0.062,
    "temperature": 0.184,
    "humidity": 0.098,
    "ph": 0.076,
    "rainfall": 0.370
  }
}

🚀 System Operational Guide

Executing via Launcher Script

Open a PowerShell console and execute:

powershell -ExecutionPolicy Bypass -File .\start_app.ps1

Manual Service Launch

Flask Service:

cd backend
pip install flask flask-cors pandas numpy scikit-learn
python app.py

Vite Service:

cd frontend
npm install
npm run dev

🧠 Explainable AI (XAI) Concepts

To ensure trust in recommendations, the system uses model transparency vectors. The application exposes feature importance coefficients to illustrate which biological and meteorological factors are key to generating the final predicted cultivation choice. This transparency is crucial for modern smart agriculture systems.
